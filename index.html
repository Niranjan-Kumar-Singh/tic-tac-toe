About the Game

Welcome to Tic Tac Toe - Play Online! This is a clean, fun, and interactive version of the classic game built using HTML, CSS, and JavaScript. Whether you want to challenge a friend or test your skills against a smart computer opponent, this game is designed for endless fun.

✨ Features

  • 🎮 Play against a friend or AI (computer)
  • 🌗 Toggle between Dark and Light Mode
  • 🎉 Confetti celebration on winning
  • 📊 Scoreboard to keep track of wins
  • 📴 Works offline (PWA supported)
  • 💻 Fully responsive and mobile-friendly

🕹️ How to Play

Take turns marking the spaces in a 3×3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the round. Switch between single-player and two-player modes anytime.

🔧 Built With

This game was developed using:

  • 💡 HTML5 for structure
  • 🎨 CSS3 for styling and themes
  • ⚙️ JavaScript for interactivity and AI logic
  • 🛠️ Service Workers for offline support
